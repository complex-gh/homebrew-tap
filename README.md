# Complex-gh Seedify

## How do I install these formulae?

`brew install complex-gh/seedify/<formula>`

Or `brew tap complex-gh/seedify` and then `brew install <formula>`.

Or, in a `brew bundle` `Brewfile`:

```ruby
tap "complex-gh/seedify"
brew "<formula>"
```

## Documentation

`brew help`, `man brew` or check [Homebrew's documentation](https://docs.brew.sh).
